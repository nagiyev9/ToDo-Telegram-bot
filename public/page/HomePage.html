Bot Commands

/help - Command list
/myTasks - Your Task List
/newTask "Task Name" "Description" expireDate - Create New Task
Example: /newTask "Task Name", "Task Description", 01-01-2025
Note: Description is Optional. If you do not want to add description write just " "
/changeStatus ID - Change task status
Example: /changeStatus 1012
/deleteTask ID - Delete Task
Example: /deleteTask 1012
